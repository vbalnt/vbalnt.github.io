Why is the Sky Blue?

physics

"Because of Rayleigh scattering" is technically correct but unsatisfying. Let's actually understand why small particles scatter blue light more than red and why sunsets are orange.

The Setup

Sunlight contains all visible wavelengths. The atmosphere is made of molecules (mostly N₂ and O₂) which are much smaller than the wavelength of light (~500 nm vs ~0.1–0.3 nm molecules).

When light hits these tiny molecules, something interesting happens: the electric field of the light wave makes the electrons in the molecule oscillate. These oscillating electrons then re-radiate light in all directions - this is scattering.

Rayleigh Scattering: The λ⁻⁴ Law

Lord Rayleigh showed in 1871 that the intensity of scattered light is:

$$I \propto \frac{1}{\lambda^4}$$

This means blue light (λ ≈ 450 nm) scatters about 6 times more than red light (λ ≈ 700 nm):

$$(700/450)^4 \approx 6$$

def wavelength_to_rgb(wavelength):
    """Convert wavelength (nm) to RGB color.
    Based on CIE color matching approximation.
    """
    if wavelength < 380 or wavelength > 780:
        return (0, 0, 0)

    if wavelength < 440:
        r = -(wavelength - 440) / (440 - 380)
        g = 0.0
        b = 1.0
    elif wavelength < 490:
        r = 0.0
        g = (wavelength - 440) / (490 - 440)
        b = 1.0
    elif wavelength < 510:
        r = 0.0
        g = 1.0
        b = -(wavelength - 510) / (510 - 490)
    elif wavelength < 580:
        r = (wavelength - 510) / (580 - 510)
        g = 1.0
        b = 0.0
    elif wavelength < 645:
        r = 1.0
        g = -(wavelength - 645) / (645 - 580)
        b = 0.0
    else:
        r = 1.0
        g = 0.0
        b = 0.0

    # Intensity adjustment at edges of visible spectrum
    if wavelength < 420:
        factor = 0.3 + 0.7 * (wavelength - 380) / (420 - 380)
    elif wavelength > 700:
        factor = 0.3 + 0.7 * (780 - wavelength) / (780 - 700)
    else:
        factor = 1.0

    return (r * factor, g * factor, b * factor)

def rayleigh_scattering(wavelength):
    """Rayleigh scattering intensity (relative)."""
    # Normalize to 550 nm
    return (550 / wavelength) ** 4

def solar_spectrum(wavelength):
    """Approximate solar spectrum (blackbody ~5778K, simplified)."""
    # Simplified: roughly flat in visible with slight peak in green
    return np.exp(-((wavelength - 500) / 200) ** 2) + 0.7

But Why λ⁻⁴?

This is where physics gets beautiful. When light hits a molecule:

  1. The oscillating electric field $E \propto \sin(\omega t)$ drives the electrons
  2. Electrons oscillate with the field, creating a tiny dipole
  3. An oscillating dipole radiates like an antenna

The radiated power from an oscillating dipole is:

$$P \propto \omega^4 |p|^2$$

where $p$ is the dipole moment and $\omega = 2\pi c/\lambda$.

Since $\omega \propto 1/\lambda$, we get $P \propto 1/\lambda^4$. (The dipole amplitude $|p|$ is roughly constant for visible light, which is well below electronic resonance frequencies.)

The key insight: higher frequency light makes electrons oscillate faster, and faster oscillation means more radiation.

Why Not Violet?

Violet has an even shorter wavelength than blue—shouldn't the sky be violet? Two reasons it appears blue:

  1. Solar spectrum: The sun emits less violet than blue
  2. Human vision: Our eyes are less sensitive to violet (the S-cones peak around 440nm, but contribute less to perceived brightness)

The result: peak perceived sky color lands around 470nm—solidly blue.

Why Sunsets Are Red

The path length through the atmosphere depends on the sun's elevation. At noon, light travels straight down through ~1 atmosphere. At sunset, it travels at a grazing angle through up to 40× more air:

$$\text{Air mass} = \frac{1}{\sin(\text{elevation})}$$

Beer-Lambert's law gives the transmitted intensity at each wavelength:

$$T(\lambda) = \exp\left(-\tau \cdot \beta(\lambda) \cdot \text{AM}\right)$$

where $\beta(\lambda) \propto 1/\lambda^4$. Blue light, with its higher scattering coefficient, is removed first. By sunset, only red survives.

fig, axes = plt.subplots(1, 2, figsize=(14, 5))

wavelengths = np.linspace(380, 700, 100)

def airmass(elevation_deg):
    """Air mass as function of sun elevation."""
    if elevation_deg <= 0:
        return 40.0
    return min(40.0, 1.0 / np.sin(np.radians(max(elevation_deg, 0.5))))

def transmitted_spectrum(elevation_deg, tau=0.08):
    """Calculate transmitted solar spectrum using Beer-Lambert law."""
    am = airmass(elevation_deg)
    # Rayleigh scattering coefficient β ∝ 1/λ⁴
    beta = (550.0 / wavelengths) ** 4
    # Transmission: T = exp(-τ × β × AM)
    return np.exp(-tau * beta * am)

def spectrum_to_rgb(spectrum):
    """Convert spectrum to RGB using color matching functions."""
    def color_match(wl, peak, width):
        return np.exp(-((wl - peak) / width) ** 2)

    r = np.sum(spectrum * color_match(wavelengths, 600, 60))
    g = np.sum(spectrum * color_match(wavelengths, 540, 50))
    b = np.sum(spectrum * color_match(wavelengths, 450, 40))

    max_val = max(r, g, b, 0.001)
    return (min(1.0, r/max_val), min(1.0, g/max_val), min(1.0, b/max_val))

Simulating the Sky

Let's build a sky renderer that captures these physics—Rayleigh scattering, Beer-Lambert transmission, and spectral integration:

# Wavelengths and Rayleigh coefficient
WL = np.linspace(400, 700, 31)
BETA = (550.0 / WL) ** 4  # Rayleigh: β ∝ 1/λ⁴
TAU = 0.05

def airmass(elev):
    """Air mass: path length through atmosphere."""
    if elev <= 0: return 40
    return min(40, 1 / np.sin(np.radians(max(elev, 0.5))))

def to_rgb(spec):
    """Spectrum → RGB via Gaussian color matching."""
    r = np.sum(spec * np.exp(-((WL-600)/60)**2))
    g = np.sum(spec * np.exp(-((WL-540)/50)**2))
    b = np.sum(spec * np.exp(-((WL-450)/40)**2))
    return np.array([r, g, b]) / max(r, g, b, 1e-6)

def sky_pixel(sun_elev, view_elev, angle_from_sun):
    """
    Compute sky color at one point.

    - Zenith: sees less-filtered sunlight → Rayleigh gives blue
    - Horizon at sunset: sees heavily-filtered sunlight → orange/red
    """
    t = view_elev / 90  # 0=horizon, 1=zenith

    # Key insight: zenith is illuminated by light from above that hasn't
    # traveled as far horizontally. Horizon sees fully-filtered sunlight.
    base_am = airmass(sun_elev)

    if sun_elev < 30:
        # Effective filtering varies: less at zenith, full at horizon
        effective_am = base_am * (0.2 + 0.8 * (1 - t))
    else:
        effective_am = base_am

    I_sun = np.exp(-TAU * BETA * effective_am)

    # Rayleigh scattering
    I_rayleigh = I_sun * BETA

    # At low sun, horizon also gets direct warm glow (haze/aerosols)
    if sun_elev < 20:
        low_sun = (20 - sun_elev) / 20
        horizon_blend = ((1 - t) ** 0.7) * low_sun

        # Warm component: heavily filtered sun (deep orange, not pale yellow)
        I_warm = np.exp(-TAU * BETA * base_am * 3.0)  # Strong filtering for rich orange
        I_sky = I_rayleigh * (1 - horizon_blend) + I_warm * horizon_blend
    else:
        I_sky = I_rayleigh

    rgb = to_rgb(I_sky)

    # Brightness: at sunset, horizon is bright (sun is there!)
    if sun_elev < 20:
        brightness = 0.75 + 0.25 * t  # Bright horizon at sunset
    else:
        brightness = 0.5 + 0.5 * t ** 0.5

    # Sun glow
    if sun_elev < 60 and angle_from_sun < 0.3:
        glow = ((0.3 - angle_from_sun) / 0.3) ** 2 * (60 - sun_elev) / 60
        sun_rgb = to_rgb(np.exp(-TAU * BETA * base_am))
        rgb = rgb * (1 - glow) + sun_rgb * glow

    return np.clip(rgb * brightness, 0, 1)

def render_sky(w, h, sun_elev, sun_az):
    """Render full sky dome."""
    sky = np.zeros((h, w, 3))

    # Sun direction vector
    se, sa = np.radians(sun_elev), np.radians(sun_az)
    sun_d = np.array([np.cos(se)*np.sin(sa), np.cos(se)*np.cos(sa), np.sin(se)])

    for y in range(h):
        ve = 90 * (1 - y/h)  # View elevation in degrees
        for x in range(w):
            va = 2*np.pi * x/w  # View azimuth
            view_d = np.array([np.cos(np.radians(ve))*np.sin(va),
                              np.cos(np.radians(ve))*np.cos(va),
                              np.sin(np.radians(ve))])
            ang = np.arccos(np.clip(np.dot(view_d, sun_d), -1, 1))
            sky[y, x] = sky_pixel(sun_elev, ve, ang)

    return sky

# Render noon and sunset
fig, axes = plt.subplots(1, 2, figsize=(14, 5))

for ax, (elev, az, title) in zip(axes, [(90, 180, 'Noon'), (5, 270, 'Sunset')]):
    sky = render_sky(300, 150, elev, az)
    ax.imshow(sky, aspect='auto', extent=[0, 360, 0, 90])
    ax.set_xlabel('Azimuth (°)')
    ax.set_ylabel('Elevation (°)')
    ax.set_title(title, fontsize=12)

plt.tight_layout()
plt.savefig(_img('sky-simulation.png'), dpi=150, bbox_inches='tight')
plt.close()

Other Atmospheres

Different planets have different atmospheres—and different sky colors:

Planet Atmosphere Sky Color Reason
Mars Thin CO₂ + dust Butterscotch/Pink Mie scattering from dust favors red
Titan Thick N₂ + haze Orange Organic haze absorbs blue
Venus Dense CO₂ Yellow-orange Thick clouds
Moon None Black No atmosphere

Earth's blue sky is not universal—it's a consequence of our particular atmosphere. A different mix of gases or particles would give us a different color overhead.
